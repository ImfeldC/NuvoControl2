
How to install ?

- Copy the ZIP file to the correct sub-folder at the path specified, under
  Tools -> Options -> 'Projects and Solutions'/General -> 'User item templates location'
  (e.g. C:\Documents and Settings\Administrator\My Documents\Visual Studio 2008\Templates\ItemTemplates )

- Done

- Now is the new template available in the create item dialog, under
  Add -> New item -> 'Visual C# Items'-> 'My Templates' (scroll at the bottom) -> 'NuvoControl.Template.CSharp'
